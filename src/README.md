# CandidateX - AI-Powered Interview Platform

A comprehensive platform for mock interviews, resume optimization, and hiring management powered by AI.

## Features

### For Candidates
- **Mock Interviews**: Practice with AI-powered interviews that adapt to your experience level
- **Real-time Feedback**: Get instant AI feedback on your responses
- **Resume Tools**: ATS-optimized resume analysis and suggestions
- **AI Assistant**: 24/7 help with interview preparation
- **Events & Workshops**: Join career development sessions
- **Progress Tracking**: Detailed analytics and performance insights

### For Recruiters
- **Resume Analyzer**: Bulk resume analysis with AI-powered candidate ranking
- **Live Interviews**: Conduct video interviews with built-in tools
- **Candidate Pipeline**: Manage and track candidates effectively
- **Analytics Dashboard**: Track recruitment metrics and performance

### For Administrators
- **System Monitoring**: Real-time system health and performance metrics
- **User Management**: Comprehensive user administration tools
- **Security Alerts**: Monitor and respond to security events
- **Analytics**: Revenue, user growth, and platform insights

## Getting Started

### Demo Accounts

Try the platform with these demo accounts:

**Candidate Account:**
- Click "Candidate Demo" on the login page
- Explore mock interviews, resume tools, and more

**Recruiter Account:**
- Click "Recruiter Demo" on the login page
- Access resume analyzer and candidate management tools

### User Flows

1. **Landing Page** → Register/Login
2. **Candidate Flow**: Dashboard → Mock Interview Setup → Pre-Check → Interview Session → Summary
3. **Recruiter Flow**: Dashboard → Resume Analyzer → Schedule Interviews
4. **Admin Flow**: Dashboard → User Management → System Monitoring
